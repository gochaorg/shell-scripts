Режим ввода/вывода.
Значение:
  • "r" - Для чтения
  • "w" - Для записи
  • "a" - Для записи в конец файла.
{string} format=optional
Формат файла (ASCII/UNICODE).
Возможные варианты:
  • "def" | "d" - По умолчанию
  • "ascii" | "a" - Ascii
  • "unicode" | "u" - Unicode.
